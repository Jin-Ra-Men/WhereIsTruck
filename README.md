#WhereIsTruck 🚚
실시간 위치 기반 노점상 및 푸드트럭 위치 공유 플랫폼입니다. 사업자 등록이 어려운 노점 사장님들과 맛있는 길거리 음식을 찾는 사용자를 신뢰 기반으로 연결합니다.

📌 프로젝트 개요
서비스명: WhereIsTruck (어디야트럭)

목적: 불확실한 노점상의 영업 여부와 위치를 실시간으로 확인하여 사용자의 헛걸음을 방지하고, 사장님들의 매출 증대를 돕습니다.

핵심 가치: 실시간성, 신뢰도 기반 검증, 지역 커뮤니티 활성화

✨ 주요 기능
👤 사용자 (User)
내 주변 트럭 찾기: 지도 API를 활용해 현재 내 위치 주변의 활성화된 노점 확인.

단골 트럭 알림: '찜'한 트럭이 영업을 시작하면 푸시 알림 수신.

실시간 리뷰 및 인증: "지금 영업 중이에요" 버튼을 통한 실시간 정보 갱신 기여.

메뉴 및 가격 확인: 사장님이 등록한 메뉴판과 사진 미리보기.

👨‍🍳 사장님 (Owner)
원클릭 영업 시작: 현재 위치 기반으로 지도에 즉시 노출 시작.

신뢰 기반 계정 인증: 현장 사진 및 사용자 추천 시스템을 통한 사장님 권한 획득.

실시간 공지: "재료 10개 남았습니다!" 등 실시간 상태 업데이트.

방문 통계: 요일별/시간별 방문자 수 및 찜 수치 확인.

🛠 기술 스택 (예시)
Frontend: React Native / Flutter (모바일 중심)

Backend: Node.js (NestJS) / Python (FastAPI)

Database: PostgreSQL (PostGIS 활용 위치 쿼리)

Real-time: Socket.io (실시간 위치 업데이트)

Infrastructure: AWS / Google Cloud Platform

🗺️ 시스템 아키텍처
위치 서비스: 사용자의 위경도 좌표를 기반으로 반경 n km 내의 트럭 필터링.

인증 시스템: 사진 메타데이터(EXIF) 검증 및 커뮤니티 추천 로직.

비즈니스 모델: 프리미엄 노출 권한 및 실시간 홍보 푸시 기능.
