형식:
- **일시:** YYYY-MM-DD (또는 HH:mm)
- **상황:** 어떤 작업 중 발생했는지
- **증상/메시지:** 에러 메시지 또는 증상
- **원인:** (알게 된 원인)
- **해결:** (적용한 해결 방법)
